# 踩点上班 App - 更新说明

## 🎉 v2.0 重大更新 - 上海地铁时刻表功能

### ✨ 新功能

#### 1. 🚇 上海地铁时刻表模块
- **支持线路**：1-18 号线完整覆盖
- **智能站点选择**：选择线路后自动加载该线路所有站点
- **实时计算**：根据起点和终点自动计算：
  - 经过站数
  - 预计乘车时间（站数 × 平均站间时间）
- **首末班车参考**：显示该线路首末班车发车时间

#### 2. 📥 一键填入功能
- 计算完地铁时间后，点击"填入地铁时间到路段配置"按钮
- 自动将地铁信息添加到通勤路段中
- 如已有地铁路段，则自动更新时间和描述

#### 3. ⇆ 交换起终点
- 点击起终点之间的交换按钮快速对调站点
- 方便查看往返时间

### 📊 数据来源
- **上海地铁官网**：https://service.shmetro.com/hcskb/index.htm
- **平均站间时间**：
  - 1-15 号线：约 2.5 分钟/站
  - 5 号线、16 号线：约 3-4 分钟/站（站间距较长）
  - 17 号线：约 3.5 分钟/站

### 🔧 技术改进
- **代码重构**：将地铁数据、应用逻辑分离为独立 JS 文件
- **文件结构**：
  ```
  commute-timer/
  ├── index.html       # 主页面
  ├── metro-data.js    # 地铁线路数据
  └── app.js           # 应用逻辑
  ```

### 💡 使用示例

1. **选择地铁线路**：如"2号线"
2. **选择起点站**：如"虹桥火车站"
3. **选择终点站**：如"世纪大道"
4. **查看计算结果**：
   - 站数：8 站
   - 预计时间：20 分钟
   - 首班车：05:25 | 末班车：22:30
5. **点击填入按钮**：自动添加到通勤路段配置

### 🎯 保持的原有功能

- ✅ 多路段通勤配置（步行、等待、红绿灯等）
